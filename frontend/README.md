##Get Started

```
$ git clone https://github.com/coala/landing.git
$ cd landing
$ git checkout hemangsk/flask-api
$ bower install
$ pip install -r requirements.txt
$ export FLASK_APP=app.py
$ flask run
```
Hooray! The site is up and running at http://localhost:5000
